This is a great entry point for any illustrator looking to go digital. Get an overview of the ADOBE Cloud platform as learners are guided through the process of logging into the ADOBE Creative Cloud, downloading the applications, and accessing cloud storage. This course begins in a lecture-intensive format with screencast explanations that familiarize learners with the ADOBE Creative Cloud graphic user interface before delving into using the software to illustrate. With a thorough tour that includes customizing workspaces and understanding windows, learners are given strategies to help learners independently problem-solve with the software in the post stages of completing this online course. This course is meant as a reference to be accessed after completion if need be. Learners are provided with resources from the ADOBE sites that assist learners in syncing libraries across ADOBE applications and creating efficient workflows. The final three units of this online course include interactive vector illustration assignments that focus on drawing with basic shapes, complex paths and anchors, and even text. This course, Introduction to ADOBE Illustrator, gives learners a starting point with the ADOBE Creative Cloud. Units are designed to build confidence in learners as they work through scaffolded activities that yield long-term skill retention.

What you’ll learn

  • Navigate the ADOBE Creative Cloud Interface and Share Content
  • Draw and edit vector shapes in ADOBE Illustrator
  • Create and manage color swatches in ADOBE Illustrator
  • Create and edit text in ADOBE Illustrator

Are there any course requirements or prerequisites?

  • It is suggested that learners have access to ADOBE Illustrator so that they may follow along with the tutorials by practicing with hands-on activities.

Who this course is for:

  • Beginners who are interested in learning to work in ADOBE Illustrator
